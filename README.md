 Project Overview
This project analyzes employee data from ABC Company using Python. The goal is to understand workforce distribution, salary structure, and relationships between employee attributes.

1) Preprocessing
  1. Replaced incorrect height values with random numbers between 150 and 180
  2. Handled dataset formatting and ensured data consistency

2)Analysis Performed
1. Employee distribution across teams
2. Employee segregation by position
3. Identification of predominant age group
4. Salary expenditure by team and position
5. Correlation between age and salar

3) Visualizations Used
1.Countplot (Team, Position, Age Group)
2.Bar Plot (Salary by Team and Position)
3.Scatter Plot (Age vs Salary)

4)Insights
1.SG position has the highest number of employees
2. Most employees belong to 20–30 age group
3. Cleveland Cavaliers have highest salary expenditure
4. Centers (C) have highest salary contribution
5. Salary increases slightly with age and experience

5) Tools Used
1.Pandas
2.NumPy
3.Matplotlib
4. Seaborn
